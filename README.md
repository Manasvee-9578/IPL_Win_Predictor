# 🏏 IPL Win Probability Predictor

A real time machine learning based web application that predicts the win probability of IPL matches based on live match conditions.

---

## 🚀 Features

* 📊 Real-time win probability prediction
* 🔴 Live match integration (API-based)
* 🎮 Interactive match simulator
* 📈 Dynamic probability graph (Plotly)
* 🌐 Full-stack app (FastAPI + Streamlit)

---

## 🧠 Tech Stack

* **Frontend:** Streamlit
* **Backend:** FastAPI
* **Machine Learning:** Scikit-learn
* **Visualization:** Plotly
* **Language:** Python

---

## 🧪 Model Details

* Trained on IPL historical data
* Features include:

  * Runs
  * Wickets
  * Overs
  * Run rate
  * Venue
